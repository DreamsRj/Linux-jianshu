# 正则表达式

java 在1.4就提供了对于支持正则表达式的包 `java.util.regex`

在这个包中，包含了两个类`Pattern`和`Matcher`类。

- Pattern 类用来表达和陈述所要搜索模式的对象
- Matcher 类是真正影响搜索的对象
- PatternSyntaxException 当遇到不合法的搜索模式时抛出
